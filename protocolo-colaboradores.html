    Protocolo de Relacionamento com Colaboradores

    Este documento estabelece o fluxo de colaboração entre pesquisadores parceiros e o staff de modelagem e escrita para produção das propostas. Versão: 1.1 — 28/06/2026

    1. Atribuições

    Staff de Modelagem e Escrita

    Função Atribuições
    Modelagem Estratégia, direcionamento técnico, integração com parceiros, aprovação de conteúdo
    Processamento e Redação Transcrição de áudios, redação, formatação, pesquisa de referências, consolidação de documentos

    O staff de modelagem e escrita não substitui o conhecimento técnico-científico dos parceiros — processa e estrutura o material fornecido, com atribuição ao autor original.

    Colaboradores (Pesquisadores, ICTs, OSPs)

    Papel Atribuições
    Pesquisador Responsável (PR) Coordena o projeto, submete a proposta no SIGCONFAP
    Pesquisador Associado (PA) Fornece dados de campo, artigos de referência, revisão crítica
    OSP Representa a comunidade, fornece diagnóstico territorial e anuência

    2. Envio de Material

    Procedimento padrão:

    1. Identificar-se no início de cada áudio: "Pesquisador [nome], ref. [nome do projeto]"
    2. Nomear o projeto — mencionar o nome do edital para alocação correta na triagem
    3. Um assunto por áudio — temas separados (orçamento, metodologia, cronograma) em arquivos distintos
    4. Acompanhar documentos — enviar PDF/link com indicação do que extrair

    Canais:

    Canal Uso
    WhatsApp Áudios curtos, links, documentos
    Telegram Transcrição automática de áudio via gateway
    GitHub Documentos consolidados, versões finais
    E-mail Documentos formais, cartas de anuência

    3. Fluxo de Trabalho

    Fase Atividade Responsável
    1. Triagem Receber material, processar (transcrever, extrair, organizar) Staff
    2. Estruturação Transformar em documento .md, alocar na pasta correta Staff
    3. Revisão Disponibilizar link para revisão Pesquisador
    4. Ajustes Incorporar correções Staff
    5. Finalização Proposta pronta para submissão Pesquisador Responsável

    4. Protocolo-base do MQTF

    Este projeto adota o protocolo-base de governança estabelecido pelo MQTF (Mulheres Que Tecem a Floresta). Trabalhos de campo que envolvam comunidades parceiras só podem ser iniciados após observância das diretrizes de consentimento, proteção de dados e soberania territorial definidas neste protocolo.
